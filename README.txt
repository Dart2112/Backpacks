Commands:
/backpack (Size) (Number)
Shows that numbered inventory
/backpack (size) (number) (Player Name)
Shows that players numbered inventory, requires backpacks.see permission and backpack.edit permission to edit the inventory

permissions are
backpacks.small.#
backpacks.medium.#
backpacks.large.#

You will find the compiled .jar in the target directory, use Backpacks.jar
